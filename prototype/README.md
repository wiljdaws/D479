# Taniti Tourism Website — Interactive Prototype

## Prototype URL

To submit this prototype as a URL link (required for Part E), host it using one of these options:

### Option 1: GitHub Pages
1. Create a new GitHub repository
2. Push the contents of the `prototype` folder to the repo
3. Enable GitHub Pages: Settings → Pages → Source: main branch
4. Your URL will be: `https://[username].github.io/[repo-name]/`

### Option 2: Netlify
1. Sign up at netlify.com
2. Drag and drop the `prototype` folder into Netlify's deploy area
3. Netlify will provide a live URL

### Option 3: Local Testing
Open `index.html` in a web browser to test locally. For submission, you must provide a hosted URL.

## Prototype Contents

- **Homepage** — Welcome, quick links to main sections
- **Lodging** — Hotels, B&Bs, resort, hostel with booking links
- **Activities** — Beaches, rainforest, volcano, entertainment (organized by type)
- **Dining** — Restaurants and grocery stores
- **Transportation** — Getting to/around the island
- **Plan Your Trip** — Currency, power, language, healthcare, holidays

## Mobile Responsive

The prototype includes a hamburger menu for mobile view (screens under 600px wide).
